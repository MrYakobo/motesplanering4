# User Stories — Mötesplanering

## Roller
- **Admin** — full åtkomst, hanterar allt
- **Medlem** — ser sitt eget schema, slides, söndag
- **Besökare** — skrivskyddad vy utan kontaktuppgifter

---

## Händelser

- Som admin vill jag skapa, redigera och ta bort händelser så att kalendern alltid är aktuell.
- Som admin vill jag se händelser i månads-, vecko-, års- och listvy så att jag kan välja den överblick som passar.
- Som admin vill jag navigera snabbt mellan veckor/månader med tangentbord (j/k) och knappar.
- Som admin vill jag söka bland händelser för att snabbt hitta en specifik.
- Som admin vill jag dra och släppa händelser i veckovyn för att ändra tid eller dag.
- Som admin vill jag propagera ändringar till framtida händelser med samma titel.
- Som admin vill jag generera återkommande händelser automatiskt.

## Schema & Tilldelning

- Som admin vill jag tilldela kontakter eller team till uppgifter per händelse i schemavyn.
- Som admin vill jag auto-fördela personer/team round-robin över framtida händelser.
- Som medlem vill jag se mitt personliga schema med mina tilldelade händelser.

## Team

- Som admin vill jag skapa team, lägga till/ta bort medlemmar och ta bort team.
- Som admin vill jag se team grupperade per uppgift och snabbt byta mellan uppgifter.
- Som admin vill jag dra och släppa personer mellan team.
- Som medlem vill jag gå med i eller lämna team själv.

## Kontakter & Konto

- Som admin vill jag hantera kontakter (namn, e-post, telefon) som kan tilldelas händelser.
- Som medlem vill jag redigera mina egna kontaktuppgifter (namn, e-post, telefon).
- Som medlem vill jag se andra medlemmars namn men inte deras e-post eller telefon (GDPR).

## Prenumeration & Kalender

- Som användare vill jag prenumerera på en kategori via iCal-feed så att händelser dyker upp i min kalender.

## Utdata

- Som admin vill jag visa en fullskärms-slideshow med veckans händelser för lobbyskärmar.
- Som admin vill jag generera och publicera ett månadsblad via SFTP.
- Som admin vill jag skicka påminnelsemail till tilldelade personer inför händelser.
- Som admin vill jag visa namnskyltar per uppgift i fullskärm.
- Som admin vill jag se en söndagsöversikt med dagens händelser och tilldelningar.

## Inställningar & System

- Som admin vill jag konfigurera organisation, SMTP, SFTP och cron-jobb i appen.
- Som admin vill jag byta accentfärg/tema för hela gränssnittet.
- Som admin vill jag kunna logga in som en annan användare för att testa deras vy.
- Som besökare vill jag se slides, namnskyltar och söndagsvy utan att logga in.
